## Project setup

```bash
$ npm install
```

## Setup .env

```
SALT_HASH = "encryption salt for login"
DB_URL_LOCAL = "mongodb://URL_LOCAL:PORT"
DB_URL = "mongodb+srv://USER:PASSWORD@URL/admin?replicaSet=replicaset&tls=true"
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
